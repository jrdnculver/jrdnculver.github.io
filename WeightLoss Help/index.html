WeightLoss Pathway Help

Login

  • There are two options with the login page
  • If you do not have an account, you can create an account with the "Create Account" Button
  • If you do have an account, you can enter your credentials and press "Sign In" Button

Create Account

  • These steps are to create an account for use with WeightLoss Pathway Application

    Name and Birthdate

    1. Enter your first name
    2. Enter your last name
    3. Use the date selector to select your birthday

  • Address and Phone Number

    1. Enter your address
    2. Enter your phone number

  • Email and Password

    1. Enter an email
    2. Create a password

Main

  • The main page is where you are able to access the entirety of the application
  • Set New Goals
  • View Goals
  • Learn More
  • Contact Us
  • Settings
  • Help
  • Sign Out

Set New Goal

  • This is where a user can select goals and create a plan

  • Personal Goal

  • If wanting to create your own goal, you can select "Personal Goal"
    • If creating a personal goal, you can create a nutritional or fitness goal
    • Remember to select the date for your goal
    • Once the information has been filled, submit goal and return to the previous page

  • Select Goal

  • From the select goal page, select a goal
  • Upon selection, you will be able to create a plan and submit the goal.
  • After goal selected and planned, you can view this goal from the View Goals page

View Goals

  • Goals can be viewed from here
  • Week of goals will load for current week
  • New Date can be selected and the week for this date will be loaded into the tabs
  • Clicking each tab once loaded will load the specific day of goals
  • Goals can be viewed on a specific date once it has been added from the Set Goals page
